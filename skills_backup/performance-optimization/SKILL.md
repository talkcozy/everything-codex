# Performance Optimization 技能

系统性能分析与优化指南。

## 性能指标

### 核心指标
- **延迟 (Latency)**: 响应时间
- **吞吐量 (Throughput)**: 每秒请求数
- **资源使用**: CPU、内存、IO
- **错误率**: 失败请求比例

### 目标值
| 指标 | 优秀 | 可接受 | 需优化 |
|------|------|--------|--------|
| API 响应 | < 100ms | < 500ms | > 500ms |
| 页面加载 | < 1s | < 3s | > 3s |
| 数据库查询 | < 10ms | < 100ms | > 100ms |

## 优化策略

### 前端优化
```
1. 代码分割
   - 路由懒加载
   - 组件按需加载
   
2. 资源优化
   - 图片压缩/WebP
   - 字体子集化
   - 资源预加载
   
3. 缓存策略
   - Service Worker
   - LocalStorage
   - CDN 缓存
```

### 后端优化
```
1. 数据库
   - 索引优化
   - 查询优化
   - 连接池
   - 读写分离
   
2. 缓存
   - Redis 缓存
   - 应用缓存
   - 响应缓存
   
3. 并发
   - 异步处理
   - 连接池
   - 批量操作
```

## 优化流程

```
1. 测量 - 确定基准
2. 分析 - 找到瓶颈
3. 优化 - 实施改进
4. 验证 - 确认效果
5. 监控 - 持续跟踪
```
