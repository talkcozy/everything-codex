# 重构 / Refactor

系统化改进代码质量。

## 使用方式

```bash
# 重构特定文件
codex /refactor src/utils.js

# 重构特定函数
codex /refactor "extract function from src/auth.js login method"

# 大规模重构
codex /refactor "refactor authentication module to use repository pattern"
```

## 重构类型

### 代码级重构
- **提取函数** - 拆分大函数
- **内联函数** - 消除过度抽象
- **重命名** - 提高可读性
- **移动方法** - 改善职责分配
- **替换算法** - 优化实现

### 结构级重构
- **提取模块** - 拆分大文件
- **合并重复** - 消除重复代码
- **简化条件** - 优化逻辑
- **引入设计模式** - 改善架构
- **依赖注入** - 解耦组件

## 安全原则

1. **测试先行** - 确保有测试覆盖
2. **小步前进** - 每次只做一项重构
3. **频繁验证** - 运行测试确认正确
4. **版本控制** - 使用 Git 便于回退
